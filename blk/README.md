# Sample Hardhat Project

To run this project locally, you need to install Hardhat and set up a local network.

```shell
npm install
npx hardhat node
npx hardhat compile
npx run scripts/deploy.js --network localhost

```

# ```shell

# npx hardhat help

# npx hardhat test

# REPORT_GAS=true npx hardhat test

# npx hardhat node

# ```
